#万分計 Linux 版
シンプルなカウントダウンタイマーです。  
何分間のカウントでも、秒針の一周で表現します。

コンパイルするには `make` とタイプして下さい。
その際、gtk2.0 の開発環境が必要となります。
また、アイコン作成に imagemagick が必要です。

##使い方
最初の画面で、何分間のカウントダウンかを指定します。  
OK ボタンをクリックまたは Enter キーの押下で、カウントダウンを開始します。  
秒針が一周したらタイムアップです。  
タイムアップでも画面は変化せず、表示が 0 になるだけです。
